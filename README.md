# ACKO Design Hiring

An AI-powered interview preparation tool for ACKO's design hiring team.

## What it does

- Upload a candidate's resume (PDF or text) and get a structured evaluation
- Covers all three interview rounds: Screening Call, Portfolio Review, and Whiteboarding
- Proposes team fit across Auto, Health & Life, Growth, Enterprise, and Labs
- Surfaces candidate-specific probe questions, prerequisites, and red flags
- Generates a shareable handoff document for the next interviewer
- Includes a Hiring Guide with the full interview process and stage-by-stage guidelines

## Built with

- React + Vite
- Anthropic Claude API (claude-sonnet-4-6)

## Setup

```bash
npm install
npm run dev
```

Requires an Anthropic API key configured in the app.
